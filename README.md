## Overview
**RenderLab** is a premium gallery of shaders and interactive visuals built with **GLSL**, **WebGL**, and **React Three Fiber**. It showcases creative coding experiments like water shaders, raymarching effects, and procedural visuals ‚Äì all running live in the browser. This is a **demo-only project**, made for exploring shaders online (not for local setup).
